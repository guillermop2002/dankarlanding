# Annonia Landing Page

Landing page estática de [Annonia](https://annonia.com) — Software de nutrición para dietistas.

## Cómo trabajar

Es un único archivo `index.html` con:
- **Tailwind CSS** via CDN (no necesitas instalar nada)
- **Lucide Icons** via CDN
- **Inter** font de Google Fonts
- Animaciones de scroll reveal con IntersectionObserver

Para verlo en local, abre `index.html` en el navegador. No necesitas servidor.

## Estructura

- `index.html` — Landing completa (navbar, hero, showcase, features, pricing, FAQ, CTA, footer)

## Notas

- Los links (`/registro`, `/login`, etc.) apuntan a la app principal en producción
- El hero tiene un placeholder "Falta imagen" — hay que poner una imagen real
- Los colores usan la paleta verde de Annonia (green-600 como primario)
